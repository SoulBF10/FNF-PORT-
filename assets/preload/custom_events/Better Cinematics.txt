Value 1: SPEED
	Mark this with any number to change how fast the
	bars move In/Out in seconds.
	(Minimum is 0.2).
Value 2: DISTANCE
	Mark this with how far you want to bars to move in.
	(Recommended 120, WOULD NOT RECOMMEND ABOVE 180, 
	but it would be kinda funny)
	(If marked with 0, then the event will end).
